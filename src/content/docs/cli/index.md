The `beaver` CLI scaffolds new projects, generates modules, and manages
GoBeaver-flavored apps.

## Status

**Alpha.** Commands and flags will change. Pin a version once you start
relying on it in CI.

## Generating reference docs

The CLI's command reference is generated from the binary itself and synced
into this site. See the `cli/` section for the per-command pages.
