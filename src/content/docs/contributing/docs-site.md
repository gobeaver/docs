This site is built with [Astro Starlight](https://starlight.astro.build/).
You don't need Node installed locally — everything runs in Docker.

See the project's [`README.md`](https://github.com/gobeaver/docs) for the
exact commands, or copy them from the root of the `docs/` directory in this
repo.
